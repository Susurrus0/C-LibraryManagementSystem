# C-LibraryManagementSystem

Library Management System demo project written in C.

Simple simulation of a library.

The following functionalities are available:

- Displaying all books in the library's catalogue

- Adding new books to the library's catalogue

The following functionalities are currently WIP:

- Deleting existing books from the library's catalogue

- Lending books from the library's catalogue to imaginary readers

- Displaying all books currently being borrowed
